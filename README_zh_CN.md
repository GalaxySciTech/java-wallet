#### 全套系统架构情况：
- 使用springboot框架
- 语言为java,kotlin
- 使用rabbitmq消息队列
- mysql云数据库
- xxl-job分布式定时任务框架

#### 全套系统均由我一个人开发，并且已经在生产环境跑了上亿流水，具有充足的可靠性，可扩展性，实用性，可以放心使用。
